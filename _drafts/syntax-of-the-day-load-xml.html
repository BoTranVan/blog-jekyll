Another one of those commands lying out there for years and I never noticed:

LOAD XML INFILE is for XML what LOAD DATA INFILE is for CSV. So you can load XML files directly into tables.

bash$ mysql sakila -e "select * from actor" --xml > /tmp/actor.xml

mysql [sakila]> create table actor_xml like actor;
Query OK, 0 rows affected (1.06 sec)

mysql [sakila]> load xml infile '/tmp/actor.xml' into table actor_xml;
Query OK, 200 rows affected (0.15 sec)
Records: 200  Deleted: 0  Skipped: 0  Warnings: 0

Pretty straightforward.
